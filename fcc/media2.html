CS50 Introduction to Computer Science

A free, entry-level course from Harvard University that teaches the foundations of computer science and programming. Learn how to think algorithmically and solve problems efficiently using languages like C, Python, and JavaScript. Perfect for beginners and curious minds.

Week 0 - Scratch

An introduction to programming fundamentals using Scratch, a visual language that teaches logic, conditions, loops, and events.

Week 1 - C

Learn the basics of C programming: variables, loops, conditionals, and functions—building the foundation for understanding how code works at a low level.

Week 2 - Arrays

Dive deeper into C with arrays and strings. Learn how to manipulate and store data efficiently.

Week 3 - Algorithms

Explore sorting and searching algorithms like binary search, selection sort, bubble sort, and understand algorithmic thinking and efficiency.

Week 4 - Memory

Understand how memory works: pointers, dynamic memory allocation, and how computers handle data at a lower level.

Week 5 - Data Structures

Learn about linked lists, stacks, queues, and trees—essential data structures that organize and manage data effectively.

Week 6 - Python

Transition to Python and experience how a high-level language simplifies coding. Learn clean, readable syntax and new problem-solving approaches.

Week 7 - SQL

Discover databases and how to use SQL to store, query, and manage structured data.

Week 8 - HTML, CSS, JavaScript

Learn the building blocks of the web: structure pages with HTML, style them with CSS, and add interactivity using JavaScript.

Week 9 - Flask

Use Flask, a Python web framework, to build dynamic web applications. This week sets the foundation for your final project.
